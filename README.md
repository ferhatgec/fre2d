# fre2d (fresh renderer enhanced 2d)
## OpenGL simple 2D renderer

## features:
* Font rendering via FreeType
* Custom framebuffer support
* Built-in orthographic camera.
* Uses DSA and non-DSA APIs using OpenGL 4.5... That's it.
